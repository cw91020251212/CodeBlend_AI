# CodeBlend AI

CodeBlend AI 是一款智能 HTML 代碼融合工具，專為前端開發者設計。它可以自動從 AI 回覆中提取 HTML/CSS/JS 代碼，並智能地替換或注入到你的原始檔案中。

## 主要功能
- **智能分析替換**：自動識別並替換現有的 HTML 標籤。
- **代碼注入**：自動將 <style> 注入 head，<script> 注入 body 結尾。
- **即時預覽**：支持桌面與手機模式預覽。
- **健康檢查**：自動檢查代碼語法錯誤（如未閉合標籤）。
- **黑夜模式**：支持流暢的日夜主題切換。
- **多語言支持**：支持中文與英文介面。

## 安裝與使用
只需下載並在瀏覽器中打開 `index.html` 即可使用。
